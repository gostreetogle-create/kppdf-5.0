## 1. Стратегия объединения

**Основа:** v5.0 (Next.js 16 + Prisma + SQLite) — современный стек, правильная архитектура.
**Наполнение:** v4.0 (Angular + Express + MongoDB) — богатый функционал, который нужно перенести и улучшить.

<a id="heading-3"></a>
### Почему v5.0 основа, а не v4.0?
- Next.js 16 (React 19) — современный, активное развитие
- Serverless API routes — не нужен отдельный backend-сервер
- Prisma ORM + SQLite — проще и быстрее MongoDB, типобезопасно
- Tailwind CSS — гибкая кастомизация под бренд
- Уже заложена правильная структура: CRUD-компоненты, Prisma schema, UI-kit

<a id="heading-4"></a>
## 2. Сравнение архитектур

| Аспект | v4.0 (Angular) | v5.0 (Next.js) |
|---|---|---|
| **Frontend** | Angular 21 + PrimeNG | React 19 + Tailwind |
| **Backend** | Express.js (отдельный) | Next.js API routes (serverless) |
| **Database** | MongoDB (Mongoose) | SQLite (Prisma ORM) |
| **Auth** | JWT + bcrypt | JWT + bcrypt |
| **PDF** | jspdf + html2canvas | ❌ Нет |
| **UI Kit** | PrimeNG (тяжёлый) | Кастомные Tailwind-компоненты (лёгкие) |
| **CRUD** | Кастомная фабрика | crud-page.tsx (уже есть) |

## 3. Функционал: что переносим из v4.0 в v5.0

<a id="heading-6"></a>
### Приоритет 1 (Критичный — ядро CRM):
- [ ] **Proposal Showcase ("Оформление КП")** — витрина товаров слева, сборка КП справа, подстановка шаблонов, PDF preview. Самая важная фича.
- [ ] **PDF генерация документов** — КП, договоры, счета. jspdf или html2canvas + jsPDF.
- [ ] **Визуальный редактор шаблонов документов** — Drag&Drop блоков (текст, таблица, разделитель), настройка высоты, полей.

<a id="heading-7"></a>
### Приоритет 2 (Важный — полнота CRM):
- [ ] **Гантт-чарт для производства** — drag&drop задач, масштаб день/неделя/месяц, фильтры по статусам, группировка по типам работ.
- [ ] **Шаблоны таблиц** — кастомные колонки для документов (JSON-конфиг колонок).
- [ ] **Финансовый модуль** — закрытие заказов, акты сверки, финансовые отчёты.
- [ ] **Складской дашборд** — остатки, заявки на закупку, заказы поставщикам, входящие счета.

<a id="heading-8"></a>
### Приоритет 3 (Улучшение — расширение функционала):
- [ ] **Маржа и анализ наценок** — расчёт наценок, визуализация.
- [ ] **Мониторинг/дашборд админа** — статистика по системе.
- [ ] **App Guide** — встроенное руководство пользователя.
- [ ] **CAD-файлы, сертификаты, RPP** — учёт конструкторской документации.
- [ ] **UI-Kit демонстрация** — страница для review компонентов.
